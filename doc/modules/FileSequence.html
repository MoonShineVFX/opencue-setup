FileSequence package¶

Module contents¶

Top-level module of FileSequence.

FileSequence contains helper classes for representing a job’s frame range.

Submodules¶

FileSequence.FrameRange module¶

Helper class for representing a frame range.

It supports a complex syntax implementing features such as comma-separated frame ranges, stepped frame ranges and more. See the FrameRange class for more detail.

class FileSequence.FrameRange.FrameRange(frameRange)¶

Represents a sequence of frame numbers.

get(idx)¶

Gets an individual entry in the sequence, by numerical position.

getAll()¶

Gets the full numerical sequence.

index(idx)¶

Query index of frame number in frame set.

Returns:

int, index of frame. -1 if frame set does not contain frame.

normalize()¶

Sorts and deduplicates the sequence.
