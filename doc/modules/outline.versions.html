outline.versions package¶

Module contents¶

outline.versions is a module for versioning external Python modules.

Submodules¶

outline.versions.main module¶

Main interface functions for interacting with outline.versions.

outline.versions.main.require(module, version='latest')¶

Adds the given module/version into the repository.

outline.versions.main.unrequire(module)¶

Removes the given module from the repository.

outline.versions.main.get_version(module, default='latest')¶

Returns the currently loaded version of the given module.

outline.versions.main.set_repos(path)¶

Set the versions repository path.

outline.versions.main.get_repos()¶

Returns the repository path.

outline.versions.main.get_session()¶

Returns the singleton session instance.

outline.versions.main.set_module_repos(module, path)¶

Sets the repository path for a specific module

outline.versions.session module¶

Classes responsible for setting up an outline.versions session.

class outline.versions.session.Session¶

The Session class handles creation of the versioning session.
